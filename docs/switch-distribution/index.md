# :material-swap-horizontal: Switch Distribution

Tài liệu quản lý **Switch Distribution** (switch phân phối) — VLAN trunking, STP configuration, link aggregation (EtherChannel/LACP).

---

## Tổng quan

Switch Distribution kết nối giữa Switch Core và Switch Access, thực hiện policy-based routing, VLAN trunking, và cung cấp redundancy.
