# :material-ethernet: Switch Access

Tài liệu quản lý **Switch Access** (switch truy cập) — port security, DHCP snooping, access VLAN, 802.1X authentication.

---

## Tổng quan

Switch Access là lớp kết nối trực tiếp với end-user devices. Cấu hình đúng switch access đảm bảo bảo mật và hiệu suất cho người dùng cuối.
