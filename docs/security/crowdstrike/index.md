# CrowdStrike Falcon

Tài liệu quản lý và vận hành CrowdStrike Falcon EDR/XDR.

## Danh sách bài viết

- Cài đặt Falcon Sensor trên Windows/Linux
- Quản lý Policy trong Falcon Console
- Xử lý cảnh báo (Detection & Prevention)
- Điều tra sự cố (Threat Hunting)
- Cách ly thiết bị bị nhiễm (Host Containment)

!!! danger "Xử lý sự cố"
    Khi có cảnh báo Critical từ Falcon, cần xử lý **ngay lập tức** và thông báo cho Security Lead.
