# ISO 27001

Tài liệu tuân thủ và triển khai tiêu chuẩn ISO 27001 - ISMS.

## Danh sách bài viết

- Phạm vi áp dụng ISMS
- Đánh giá rủi ro (Risk Assessment)
- Chính sách bảo mật thông tin
- Kiểm soát tiếp cận (Access Control)
- Kế hoạch ứng phó sự cố (Incident Response)
- Audit & Review định kỳ

!!! info "Audit"
    Đánh giá nội bộ thực hiện **6 tháng/lần**, đánh giá bên ngoài **12 tháng/lần**.
