# Grafana

Tài liệu cài đặt, cấu hình và vận hành Grafana.

## Danh sách bài viết

- Cài đặt và cấu hình ban đầu
- Thêm host / agent mới
- Cấu hình cảnh báo (Alert)
- Dashboard & báo cáo
- Troubleshooting thường gặp

!!! note "Cập nhật tài liệu"
    Vui lòng cập nhật sau khi thêm host mới hoặc thay đổi cấu hình alert.
