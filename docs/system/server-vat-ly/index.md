# Server vật lý

Tài liệu quản lý và vận hành các server vật lý (bare-metal).

## Danh sách bài viết

- Kiểm tra phần cứng & trạng thái server
- Cấu hình BIOS / iDRAC / iLO
- Cài đặt OS từ đầu
- Thay thế phần cứng (RAM, HDD, PSU)
- Giám sát nhiệt độ & cảnh báo phần cứng

!!! note "Cập nhật tài liệu"
    Vui lòng cập nhật thông tin server sau khi thực hiện thay đổi phần cứng hoặc cấu hình.
