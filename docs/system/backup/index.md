# Backup & Restore

Quy trình backup và phục hồi dữ liệu hệ thống.

## Danh sách bài viết

- Quy trình backup VM (Proxmox Backup Server)
- Backup cấu hình thiết bị mạng
- Backup dữ liệu File Server
- Quy trình Restore khẩn cấp

!!! success "Nguyên tắc 3-2-1"
    Luôn duy trì **3 bản sao**, trên **2 loại media khác nhau**, trong đó **1 bản offsite**.
