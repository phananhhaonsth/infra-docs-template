# :material-desktop-tower: Server vật lý

Tài liệu quản lý **server vật lý** — BIOS/UEFI, RAID configuration, iDRAC/iLO remote management, bảo trì phần cứng.

## Tổng quan

Quản lý và bảo trì server vật lý là nền tảng của hạ tầng IT. Phần này bao gồm các quy trình kiểm tra, bảo trì, và xử lý sự cố phần cứng.
