# :material-shield-lock: Firewall

Tài liệu cấu hình và quản lý **Firewall** — rules, NAT, VPN, logging, monitoring, và troubleshooting.

## Tổng quan

Firewall là lớp bảo vệ quan trọng nhất trong hạ tầng mạng. Phần này bao gồm các quy trình cấu hình, quản lý policy, và xử lý sự cố firewall.
