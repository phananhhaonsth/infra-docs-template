# :material-lan: Switch Core

Tài liệu quản lý **Switch Core** (switch lõi) — VLAN routing, inter-VLAN routing, redundancy (HSRP/VRRP), QoS, và monitoring.

---

## Tổng quan

Switch Core là thiết bị trung tâm trong kiến trúc mạng 3 lớp, chịu trách nhiệm routing giữa các VLAN và kết nối đến firewall/router.
