# Microsoft 365

Tài liệu quản lý tài khoản và cấu hình Microsoft 365.

## Danh sách bài viết

- Tạo và quản lý tài khoản
- Cấu hình MFA / 2FA
- Phân quyền và nhóm
- Quy trình onboarding / offboarding nhân viên
- Quản lý license

!!! danger "Bảo mật"
    Khi nhân viên nghỉ việc, vô hiệu hóa tài khoản trong vòng **24 giờ**.
