## :material-map-marker-path: Tổng quan

Chào mừng bạn đến với kho tài liệu vận hành nội bộ của team Infrastructure. Tài liệu này được thiết kế để giúp bạn đi từ kiến thức nền tảng đến các kỹ thuật nâng cao trong quản lý và vận hành hạ tầng hệ thống.

!!! tip "Cách sử dụng tài liệu"
    - Sử dụng **thanh tìm kiếm** (phím tắt: ++s++ hoặc ++f++) để tìm nhanh nội dung
    - Chọn **tab** trên thanh navigation để đi đến từng chủ đề
    - Mỗi trang có **Table of Contents** bên phải để điều hướng trong trang
    - Nhấn vào hình ảnh để **phóng to** xem chi tiết

## :material-account-group: Đóng góp nội dung

Mỗi thành viên trong team đều có thể đóng góp tài liệu. Xem [Hướng dẫn đóng góp](contributing.md) để biết cách viết và submit tài liệu mới.

Quy trình cơ bản:

1. **Clone** repo về máy local
2. **Tạo file** `.md` trong thư mục phù hợp
3. **Viết nội dung** bằng Markdown (hỗ trợ hình ảnh, video, bảng, code)
4. **Push** lên GitHub → tự động deploy

!!! info "Cần hỗ trợ?"
    Liên hệ admin team nếu bạn cần quyền truy cập repo hoặc hỗ trợ viết tài liệu.
